Камень
{
	если падает на огонь, то огонь исчезает
}
дерево
{
	если касается огня, то над ним появляется огонь
	если падает в воду, то всплывает на половину(пол блока под водой)
}
огонь
{
	можно поставить только ткнув на объекты, которые могут гореть (дерево, не вода, трава, не камень)
}
холод
{
	не является объектом
	при взаимодействии с огнем, тушит его
	при взаимодействии с водой, превращает в лед 
	при взаимодействии с паром, превращает в лед
}
лед
{
	нельзя просто поставить, можно только создать
	если падает на огонь, то тушит его (не тает)
	если огонь не дальше чем в 2-х блоках, то превращается в воду через 3 секунды
	в воде не тонет
	если касается воды то вода превращается в лед
}
вода
{
	установка по dfs(то есть заполняет клетку на которую ткнули и пустые клетки снизу справа и слева(сверху не надо) и так по рекурсии), если не хватает воды для установки, то не ставится
	если огонь не дальше чем в 2-х блоках, то превращается в пар через 3 секунды
}

ветер
{
	нельзя просто поставить, уже стоит на уровне
	каждая клетка ветра имеет направление
	если пар назодится в клетке с ветром, то передвигается на одну клетку в зависимости от направления
	если пар врежется в подвижный обьект, то он сдвинется
}

пар
{
	нельзя просто поставить, можно только создать
}
